# Aero Telemetry Hub

Aero is a premium, interactive aerodynamics telemetry visualizer and environmental simulation dashboard built with React and Vite.

## Project Documentation

To learn more about the project, check the following documents:
- [**Project Structure**](file:///p:/projects/Aero/STRUCTURE.md): Comprehensive file directory layouts and references.
- [**Completed Updates**](file:///p:/projects/Aero/UPDATES.md): List of implemented features, styling configuration, and codebase commits.
- [**Future Roadmap**](file:///p:/projects/Aero/ROADMAP.md): Next steps, interactive aerofoil designers, and physics updates.

## Technical Stack
- React (Single Page Application)
- Vite (Fast development build server)
- Vanilla CSS (Glassmorphism layout, animations, variables, custom fonts)
- HTML5 Canvas (Laminar flow visualizer, real-time particle dynamics, live graphs)

## Installation & Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run in development mode**:
   ```bash
   npm run dev
   ```
3. **Compile for production**:
   ```bash
   npm run build
   ```
